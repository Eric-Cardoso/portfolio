# portfolio
Portfólio contendo projetos pessoais.

## Projetos
### Automação-Alura
Automação de login na plataforma Alura utilizando Python.

- Tecnologias utilizadas: Python(POO), Selenium, PyAutoGUI, python-dotenv.

- Link: https://github.com/Eric-Cardoso/automacao-login-alura.git

### API de Mercado
API REST de um sistema de mercado com autenticação JWT, carrinho de compras e gerenciamento de produtos. O projeto simula um fluxo completo de e-commerce, com regras de negócio como descontos automáticos por categoria e controle de carrinho por usuário.

- Tecnologias utilizadas: Python, FastAPI, SQLAlchemy, SQLite, Alembic, JWT, Bcrypt.

- Link: https://github.com/Eric-Cardoso/projeto_mercado.git

### API de Fichas
API REST desenvolvida com FastAPI para gerenciamento de usuários e fichas, com foco em organização, escalabilidade e boas práticas de arquitetura. A aplicação conta com autenticação JWT, controle de permissões entre usuários e administradores, cache com Redis para otimização de desempenho e um sistema completo de gerenciamento de fichas com operações CRUD.

- Tecnologias utilizadas: Python, FastAPI, SQLAlchemy, Alembic, SQLite, Redis, JWT, Bcrypt, Pydantic, python-dotenv, MkDocs.

- Link: https://github.com/Eric-Cardoso/projeto_ficha.git
